# Claude Learner

This repo contains two Claude Code skills for interactive learning.

## Structure
- `skills/learn/SKILL.md` — the /learn skill (fetches URLs, generates curricula)
- `skills/teach/SKILL.md` — the /teach skill (interactive tutor)
- `templates/` — setup guides for Obsidian users
- `examples/` — sample curriculum output

## Contributing
- Skills are pure markdown — no code to build or test
- Test changes by copying skills to `~/.claude/skills/` and invoking them
